# TU-CIS-1051-Final-Project
Final project of CIS 1051 at Temple University


Support Vector Machine models tries to find the hyperplane that best separates two classes by maximizing the margin between the represented by a linear equation. This margin is the distance from the hyperplane to the nearest support vectors on each side.

K Nearest Neighbor models identify the closest data points to an input and makes predictions based on the average value of the data points. KNN assigns categories based on the majority of nearby points, working by proximity and majority to make predictions.

Random Forest models creates decision trees to make predictions. Each tree analyzes random parts of the data and combines them together for regression and predictions. The final prediction is the majority vote of each tree.

Each method was fairly accurate (above 70%), but RF model was most accurate at an average of 87% accuracy. SVM uses linear hyperplanes to make predictions, but this data is not linear, therefore it is not that accurate.
